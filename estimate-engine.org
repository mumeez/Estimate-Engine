* Overview

Construction & remodel estimating web app with eldritch-themed UI,
canvas particle background, Home Depot pricing database, and California
building code search. Runs entirely in the browser — no backend needed.

** Quick Start

#+BEGIN_SRC sh
cd ~/Projects/estimate-app && python3 -m http.server 8080
#+END_SRC

Then open http://localhost:8080.

** Repo

https://github.com/mumeez/estimate-engine

* Current Features

** Core Engine
- Materials database: 200+ items / 14 categories w/ Home Depot pricing
- Auto-fill materials by search
- Hourly rate slider ($15–$250/hr)
- Tax toggle, material markup %, discount %
- Simple / Advanced mode toggle

** Advanced Mode
- Overhead %, profit margin %
- Subcontractors w/ license tracking
- Equipment costs, permits/fees
- Waste factor, per-item markup, labor categories

** Quick Calculators
- Drywall, Concrete, Lumber, Paint, Flooring
- All can auto-add results to estimate

** Project Management
- Notes, status (Estimating → In Progress → Completed → On Hold)
- Crew size, start/end dates
- Change orders w/ status toggle
- Save/load to localStorage

** Code Search
- 78 CA building code refs: CRC, CBC, CALGreen, CEBC, CPC, CMC, CEC

** Export
- Obsidian Markdown (full project note w/ YAML frontmatter)
- Print / PDF via browser

** UI / Theme
- Dark eldritch palette (Dracula-inspired)
  - Background: #363748 (deep), #404262 (surface)
  - Headers: #BC93FA (purple)
  - Text: #33ff66 (green)
- Font: CaskaydiaCove Nerd Font
- Canvas particle background (stars, connector lines, drifting glyphs)
- NO CSS text animations — only background moves

* Design Decisions

| Decision | Why |
|----------+-----|
| Vanilla JS (no framework) | No build step, simple to share |
| Local JSON for materials | Works offline immediately |
| Canvas API for background | Smooth animation, no text movement |
| Python http.server | Zero deps to run locally |
| localStorage | No server needed for persistence |
